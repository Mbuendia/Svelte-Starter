## Getting Started

1. Clone this boilerplate

```bash
$ git clone --depth 1 https://github.com/Shyam-Chen/Svelte-Play.git <PROJECT_NAME>
$ cd <PROJECT_NAME>
```

2. Install dependencies

```bash
$ yarn install
```

3. Start a local server

```bash
$ yarn start
```
