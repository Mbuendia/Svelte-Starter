## 如何使用?

1) 複製底板
```bash
$ git clone --depth 1 https://github.com/Shyam-Chen/Vanilla-Starter-Kit.git <PROJECT_NAME>
$ cd <PROJECT_NAME>
```

2) 安裝相依套件
```bash
$ npm install
```

3) 執行應用程式
```bash
$ npm start
```
