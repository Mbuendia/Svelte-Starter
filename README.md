**專案主要是使用 Gulp + Rollup**

## 如何使用?

1) 安裝 [Node](https://nodejs.org/en/)

2) 下載此[樣板](https://github.com/Shyam-Chen/Front-End-Seed/archive/master.zip)

或者使用 `git clone`

3) 安裝相依套件
```bash
$ yarn install  # or npm install
```

4) 執行應用程式
```bash
$ yarn start  # or npm start
```
